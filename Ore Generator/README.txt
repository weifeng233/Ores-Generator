All ore's weight in data.json.
If your ore has data, use id:data, for example: stone:2
所有的矿石的权重都在data.json里
如果你的矿石有特殊值，请使用id:data，例如：stone:2

Example:
{
	"Minecraft": {
		"enable": true,
		"ores": {
			"gravel": 18,
			"stone:0": 30,
			"stone:1": 11,
			"stone:2": 9
		}
	},
	"ModName": {
		"enable": true,
		"ores": {
			"oreA": 12,
			"oreB:0": 15,
			"oreB:1": 5
		}
	}
}
